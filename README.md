

# Installation

Install meteor (https://www.meteor.com/install)

    curl https://install.meteor.com/ | sh

Clone this repository

    git clone git@github.com:Paratii-Video/paratii-player.git

Install the package

    cd paratii-player
    npm install

Now start the application

    npm start
